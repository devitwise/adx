# ADX — AI Development Experience

Claude Code plugin marketplace for project backlog management and codebase auditing.

## Install

```bash
# Add marketplace
/plugin marketplace add devitwise/adx

# Install plugin
/plugin install adx@adx
```

## Plugins

- **[adx](plugins/adx/)** — `/adx-init`, `/adx-sync`, `/adx-audit`, `/adx-convert`

See [plugin README](plugins/adx/README.md) for full docs.
